Colinealidad

  • El clculo de los coeficientes implica resolver \(\beta = (X'X)^{-1}(X'Y)\)
  • Si los datos estn muy correlacionados, la determinante de \((XX)\) tiende a 0. Los valores de la matriz invertida se vuelven sensibles a pequeas variaciones en los datos originales.
  • Pequeos cambios en los valores o el hecho de agregar o quitar variables provoca grandes diferencias en la estimacin de los parmetros de regresin.
  • La matriz de varianzas de los estimadores es proporcional a XX, por lo que en presencia de colinealidad los errores estndar de los coeficientes son grandes.

  • Una variable cualquiera \(X\) siempre estar fuertemente correlacionada con el trmino cuadrtico correspondiente \(X^2\). Una posible solucin para minimizar este efecto es centrar la variable X (restarle la media a cada valor, de modo que la nueva media sea 0).
  • Dos variables cualquiera \(X, Z\) siempre estarn correlacionadas con su interaccin. La colinealidad slo se examina en modelos sin interacciones.

Seleccin de modelos.

Dos objetivos:

  • Explicacin: encontrar el set de variables que sean ms im portantes para explicar la variacin en la respuesta.
  • Prediccin: la precisin de nuestras estimaciones sern mejores en un modelo con pocas variables.

Tres Mtodos

  • Best subset selection. Encontrar un set de p variables predictoras.
  • Shrinkage. Modelo con todas las p donde algunos coeficientes son reducidos a cero.
  • Dimension reduction. Proyeccin de las p variables en un espacio de menores dimensions (por ej. PCA).

  • Varianza. Cmo puede cambiar el modelo con diferentes datos de entrenamiento. Modelos menos flexibles (ms sencillos) tienen menos varianza.
  • Sesgo. Error introducido al momento de enfrentar la vida real (que puede ser extremadamente complicada), utilizando un modelo sencillo.

Best subset selection

Seleccin basada en criterios

\[AIC = -2 log(likelihood) + 2p\]

\[BIC = -2 log(likelihood) + log(n)p\]

Existe una fauna de criterios muy variada: Cp, DIC, AICc, BICc, \(R^2_{adj}\) etc. Tambin se puede utilizar validacin cruzada en vez de un criterio, que estima de forma directa el error de prueba.

Seleccin de modo forward

Al modelo slo con intercepto se le incorpora un creciente nmero de variables, hasta que la adicin no provoca mejoras en el criterio elegido. El anteltimo modelo es por lo tanto el mejor.

Seleccin de modo backward

Al modelo ms completo posible se le remueven variables hasta que la adicin no provoca mejoras en el criterio elegido. El ltimo modelo es por lo tanto el mejor.

Bsqueda exhaustiva

Se construyen todos los modelos posibles y se elige el mejor de acuerdo a cierto criterio.

Promediado de modelos

Se construyen todos los modelos posibles y se presenta el set de mejores modelos, dentro de determinado rango de criterios.

Shrinkage

\[RSS = \sum_{i = 1}^{n}(y_i - \beta_0 - \sum_{j = 1}^{p} \beta_jx_{ij})^2\]

Ridge regression

\[RSS + \lambda \sum_{j = 1}^{p} \beta_j^2\]

Lasso (least absolute shrinkage selector operator)

\[RSS + \lambda \sum_{j = 1}^{p} |\beta_j|\]

  1. Ambos estiman una suma de cuadrados penalizada.
  2. El efecto es contraer los estimadores de \(\beta_{i}\) hacia cero (excepto el intercepto).
  3. Lasso sobresale porque logra que los estimadores de algunas variables se vuelvan exactamente cero, por lo que realiza a la vez seleccin de variables y de modelos.
  4. Ambos requieren ajustar un parmetro de afinado por validacin cruzada.
